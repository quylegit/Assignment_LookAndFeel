Design Elements

  • Responsive - Design must incorporate be responsive design principals
  • Usability - User interface and navigation must be intuitive for the end user
  • Content - Content must be revelant to each other
  • Design - Design must use at least three specific design guidelines and ideas for web design

Responsive

Tablet internet consumption grew 30% between 2013 and 2015. In the same period, smartphone internet consumption grew 78%. Thus the website is designed to be responsive for large desktop users to small smartphone screens.

Usability

  • Menu can easily be located at all times.
  • Menu will change as screen size change.
  • Content and all relavant information is centered.
  • Each section is distinguishable.
  • Each page is only one page deep.

Content

Web page content is related to investing, particulary, dividend investing. Index page explains how one can start investing. Best Dividend Stocks explains the types of dividend stocks available, especially in today's volatile market. All-Time Top 6 Dividend Stocks shows the all-time dividend choices. Investing Knowledge explains a brief investing terminology.

Design

  • Color: Consistent style and color scheme throughout.
  • Font: Only two main font styles are used. Spacing used should be comfortable for reading.
  • Layout: Site uses css grids for consistent layout.The white space usage should be balanced.
  • Responsive: Site is designed to respond to both large and small screens. For large screens, the width of the content is limited for readability. Content will shift for best viewing experience for small screens.
  • Conventional: Logo at the top left; Logo and Home (icon) brings you home; Menu items highlighted when a mouse hovers over.
